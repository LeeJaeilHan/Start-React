# 16. Redux

## Redux의 장점

1. 컴포넌트 간 의존성을 줄여주어 로직을 간단하게 해준다 (중앙 집중적인 data store)
2.  시간 여행을 할 수 있다. 

## Redux 실습

- 핵심 키워드
- Parcel로 프로젝트 구성
- 토글 스위치와 카운터 구현
